# PRODIGY_GA_01

## Text Generation using GPT-2

This project demonstrates text generation using the GPT-2 model from Hugging Face Transformers. The model generates human-like text based on a given prompt.

## Internship Details
- Internship: Prodigy InfoTech
- Domain: Generative AI
- Task: GA-01
- Project: Text Generation using GPT-2

## Technologies Used
- Python
- Hugging Face Transformers
- GPT-2
- PyTorch

## Project Description
In this project, the GPT-2 pre-trained language model is used to generate meaningful text from a user prompt. The model predicts the next words in a sequence to create coherent paragraphs.

## Example Prompt
"The future of artificial intelligence is"

## Example Output
"The future of artificial intelligence is expected to transform industries, improve automation, and enhance human productivity."

## How to Run
1. Install required libraries
2. Load the GPT-2 model using Hugging Face
3. Provide a text prompt
4. Generate text output

## Outcome
Successfully generated human-like text using GPT-2 demonstrating the power of generative AI models.
